# 🍔 Burger Landing — аппетитный лендинг для фастфуда
Яркий и интерактивный лендинг для виртуального ресторана быстрого питания, созданный с использованием HTML и CSS. Дизайн полностью повторяет макет из Figma, а анимации добавляют сайту динамики и вовлекают пользователя.

## 🛠️ Использованные технологии
* **HTML5** — семантическая вёрстка для корректного отображения структуры страницы.
* **CSS3** — стилизация и визуальное оформление.
* **CSS‑анимации** — интерактивность и динамика элементов.

## ✨ Ключевые особенности проекта
* **Полная вёрстка по макету из Figma** — точное соответствие дизайнерскому прототипу, пиксель в пиксель.
* **Интерактивные кнопки с анимацией при наведении** — элементы оживают при взаимодействии, создавая приятный пользовательский опыт.
* **Адаптивный дизайн** — корректное отображение на экранах любых размеров.
* **Продуманная типографика и цветовая палитра** — визуальная гармония и читаемость контента.
